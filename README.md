### A Portfolio Web APP

A Portfolio Sample Responsive Web Application

## Getting Started

First, run the development server:

```bash
npm run build
npm run dev

# or
yarn build
yarn dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technology Stack

- Next.js
- React
- Media Query
